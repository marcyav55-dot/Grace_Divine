🔴 BLOQUÉ ACTUELLEMENT

Erreur au chargement de /admin/ :
zoneinfo._common.ZoneInfoNotFoundError: 'No time zone found with key UTC'

Cause : le module Python tzdata (fuseaux horaires) n'est pas disponible
dans l'environnement (Termux/smartIDE sur Android). La tentative
"pip install tzdata" n'a pas encore été confirmée comme résolue.

Prochaine étape à essayer :
- Relancer précisément : pip install tzdata
  (ou : python3 -m pip install tzdata)
- Puis relancer le serveur et retester /admin/
- Si ça persiste, explorer une autre piste :
  - Forcer TIME_ZONE = None temporairement dans settings.py
  - Vérifier la version de Python/pip utilisée
