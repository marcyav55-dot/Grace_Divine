## 7. Notifications push (blog)
Statut : En cours - bloque sur une dependance

Plan : cles VAPID -> endpoint PushSubscription -> mise a jour sw.js -> demande de permission frontend -> envoi auto a la publication (pywebpush).

Blocage : pip install pywebpush echoue sur Android/Termux (cryptography necessite Rust, cible aarch64-unknown-linux-android non supportee).

Solutions a tester a la reprise :
- Option A : pkg install python-cryptography puis retenter pip install pywebpush py-vapid --break-system-packages.
- Option B : generer les cles VAPID via Node (npm install -g web-push && web-push generate-vapid-keys).
